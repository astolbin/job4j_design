Придумайте 3 примера, когда происходит нарушение принципа DIP:
1. внутри метода генерации отчета - создается обьект конкретного формата отчета
2. одно из полей класса - конкретный класс контейнера обьектов (ArrayList) вместо его интерфейса (List)
3. параметр метода имеет конкретный тип хранилища обьектов (MemoryStore)
